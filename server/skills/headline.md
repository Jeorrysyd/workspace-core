# 标题生成 Skill

为内容生成 10 个标题候选，按平台优化，帮用户选出最有杀伤力的标题。

---

## 标题公式库

| 公式 | 模板 | 适合类型 |
|------|------|----------|
| 数字型 | "X 个 [主题] 的 [技巧/真相/方法]" | 技巧合集、指南 |
| 反常识型 | "[主流认知]？错了。" / "别再 [错误做法] 了" | 观点型、世界观论证 |
| 结果型 | "我用 [工具/方法] [达成了什么结果]" | Showcase |
| 悬念型 | "为什么 [现象]？" / "[现象] 的真正原因" | 深度文章 |
| 代入型 | "如果你也 [痛点]，试试 [方案]" | 指南、技巧 |
| 对比型 | "[A] vs [B]，[结论]" | 测评、观点 |

---

## 平台适配规则

### 小红书
- 标题 ≤ 20 字
- 可用 emoji 但不超过 2 个
- 口语化，像跟朋友说话
- 数字开头效果最好
- 避免："干货""建议收藏""赶紧看"（降权词）

### 公众号
- 标题可以更长（15-30 字）
- 有判断力和态度
- 适度悬念但不标题党
- 可以用破折号分层

### Twitter/X
- 标题即正文第一句
- 要有冲击力，像喊话
- 英文可混用
- 极简，每个字都有信息量
