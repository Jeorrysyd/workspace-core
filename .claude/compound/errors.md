## Node.js / Express

### [Node.js] 新增路由后 404
**症状**：添加新 API 路由后请求返回 404
**根因**：Express 后端不支持热重载，新路由需要重启服务才能生效
**解法**：每次添加新路由后 `npm start` 重启服务
**来源项目**：Workspace J
**日期**：2026-03

## 数据存储

### [JSON] 批量写入时 ID 碰撞
**症状**：`data/memory/memory.json` 出现重复 `id`（如两条都是 `mem-1772206079328`）
**根因**：`Date.now()` 在同一事件循环内循环调用时返回相同值
**解法**：用 `Date.now() + index` 或 `Date.now() + Math.random()` 生成唯一 ID
```js
// ❌ 错误
items.map(() => ({ id: `mem-${Date.now()}` }))

// ✅ 正确
items.map((item, i) => ({ id: `mem-${Date.now()}-${i}` }))
```
**来源项目**：Workspace J
**日期**：2026-03
